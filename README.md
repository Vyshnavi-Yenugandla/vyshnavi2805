# Smart Health System

A web-based Smart Health System developed using Node.js, Express, MongoDB, and HTML.

## Features
- Appointment booking
- User authentication
- Dashboard for users
- Health-related services

## Technologies Used
- Node.js
- Express.js
- MongoDB
- HTML, CSS, JavaScript

## How to Run
1. Clone the repository
2. Run `npm install`
3. Start server using `node server.js`
